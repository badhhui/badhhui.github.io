42. 接雨水

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

示例 1：

输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

示例 2：

输入：height = [4,2,0,3,2,5]
输出：9
动态规划

每个坐标能储存的雨水数量取决于当前坐标高度、左边的最高值以及右边的最高值，因此可以先计算出，每个坐标的左边的最高值和右边的最高值，分别存储在lmax和rmax数组中，然后遍历height内的每根柱子，计算当前柱子的储水量，即min(lmax[i], rmax[i]) - height[i]，把每个坐标的储水量相加即可。

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        int lmax[n], rmax[n];
        int ans = 0;
        for (int i = 0; i < n; i++) {
            if (i == 0) lmax[i] = max(0, height[i]);
            else lmax[i] = max(lmax[i-1], height[i]);
        }
        for (int i = n - 1; i >= 0; i--) {
            if (i == n - 1) rmax[i] = max(0, height[i]);
            else rmax[i] = max(rmax[i+1], height[i]);
        }
        for (int i = 0; i < n; i++) {
            ans = ans + min(lmax[i], rmax[i]) - height[i];
        }
        return ans;
    }
};
双指针

动态规划做法中使用两个数组记录了每个坐标的左右两边的最高的柱子高度，并且当前坐标储水量取决于lmax和rmax中的较小值，lmax是从左往右计算的，rmax是从右往左计算的，所以可以使用两个指针l和r分别指向height的左右两端，让l向右移动，r向左移动，并用当前l和r的柱子高度更新lmax和rmax的值，比较l和r的高度值，如果height[l]<height[r]，说明此时lmax<rmax，则当前坐标的储水量=lmax - height[l]，让l向右移；如果height[l]>=height[r]，说明此时lmax>=rmax，则当前坐标的储水量=rmax - height[r]，让r向左移，重复以上过程直到l和r相遇即可得到总储水量。

class Solution {
public:
    int trap(vector<int>& height) {
        int n = height.size();
        int lmax = 0, rmax = 0;
        int ans = 0;
        int i = 0, j = n - 1;
        while (i < j) {
            lmax = max(lmax, height[i]);
            rmax = max(rmax, height[j]);
            if (height[i] < height[j]) {
                ans = ans + lmax - height[i];
                i++;
            } else {
                ans = ans + rmax - height[j];
                j--;
            }
        }
        return ans;
    }
};

438. 找到字符串中所有字母异位词

中等

给定两个字符串 s 和 p，找到 s 中所有 p 的 异位词 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

异位词 指由相同字母重排列形成的字符串（包括相同的字符串）。

示例 1:

输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。

示例 2:

输入: s = "abab", p = "ab"
输出: [0,1,2]
解释:
起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。

提示:

  • 1 <= s.length, p.length <= 3 * 104
  • s 和 p 仅包含小写字母
滑动窗口

维护两个数组，用来保存窗口中s包含的各个字母出现的次数以及p包含的各个字母出现的次数，每次滑动窗口前，减去窗口最左边字母的个数，加上窗口右边一个字母的个数，每次滑动后判断字母出现次数是否相等。

**优化：**不维护滑动窗口和字符串 p 中每种字母的数量，而是统计滑动窗口和字符串 p 中每种字母数量的差；并引入变量 differ来记录当前窗口与字符串 p 中数量不同的字母的个数，当differ等于0时说明窗口内字母出现次数相等。

class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        int sLen = s.size(), pLen = p.size();

        if (sLen < pLen) {
            return vector<int>();
        }

        vector<int> ans;
        vector<int> count(26);
        for (int i = 0; i < pLen; ++i) {
            ++count[s[i] - 'a'];
            --count[p[i] - 'a'];
        }

        int differ = 0;
        for (int j = 0; j < 26; ++j) {
            if (count[j] != 0) {
                ++differ;
            }
        }

        if (differ == 0) {
            ans.emplace_back(0);
        }

        for (int i = 0; i < sLen - pLen; ++i) {
            cout << s[i] << endl;
            if (count[s[i] - 'a'] == 1) {  // 窗口中字母 s[i] 的数量与字符串 p 中的数量从不同变得相同
                --differ;
            } else if (count[s[i] - 'a'] == 0) {  // 窗口中字母 s[i] 的数量与字符串 p 中的数量从相同变得不同
                ++differ;
            }
            --count[s[i] - 'a'];

            if (count[s[i + pLen] - 'a'] == -1) {  // 窗口中字母 s[i+pLen] 的数量与字符串 p 中的数量从不同变得相同
                --differ;
            } else if (count[s[i + pLen] - 'a'] == 0) {  // 窗口中字母 s[i+pLen] 的数量与字符串 p 中的数量从相同变得不同
                ++differ;
            }
            ++count[s[i + pLen] - 'a'];
            
            if (differ == 0) {
                ans.emplace_back(i + 1);
            }
        }

        return ans;
    }
};

560. 和为 K 的子数组

中等

给你一个整数数组 nums 和一个整数 k ，请你统计并返回 该数组中和为 k 的子数组的个数 。

子数组是数组中元素的连续非空序列。

示例 1：

输入：nums = [1,1,1], k = 2
输出：2

示例 2：

输入：nums = [1,2,3], k = 3
输出：2

提示：

  • 1 <= nums.length <= 2 * 104
  • -1000 <= nums[i] <= 1000
  • -107 <= k <= 107
前缀和

求数组里某些连续元素的和，可以想到使用前缀和，建立前缀和数组s，然后遍历前缀和数组，可以双重循环找到前缀和数组的两个下标i和j，如果s[i]等于k，说明原数组从0到i是符合条件的，如果s[i]-s[j]=k，说明元数组从j+1到i是符合条件的。

优化方法：在遍历前缀数组时，假如现在的元素是s[i]，我们想要看i之前有没有符合条件的子数组，就需要在前缀和数组的下标i之前的元素寻找元素值为s[i]-k的下标。我们可以维护一个map，键为前缀和数组的元素值，值为这个元素到目前为止出现的次数。在遍历前缀和数组时，先判断当前元素是不是等于k，再去map找s[i]-k，如果找到了就增加map中键为s[i]-k对应的值，最后再把当前元素添加到map中。

class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        const int N = 2e4 + 10;
        int s[N] = {0};
        int cnt = 0;
        unordered_map<int, int> umap;
        for (int i = 1; i <= nums.size(); i++) {
            s[i] = s[i-1] + nums[i-1];
        }
        for (int i = 1; i <= nums.size(); i++) {
            if (s[i] == k)
                cnt++;
            if (umap.find(s[i]-k) != umap.end())
                cnt += umap[s[i] - k];
            umap[s[i]]++;
        }
        return cnt;
    }
};

239. 滑动窗口最大值

困难

给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。

返回 滑动窗口中的最大值 。

示例 1：

输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
输出：[3,3,5,5,6,7]
解释：
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7

示例 2：

输入：nums = [1], k = 1
输出：[1]

提示：

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104
  • 1 <= k <= nums.length
单调队列

维护一个单调队列，在滑动窗口时将新进入窗口的值与从队尾到队首每个元素比较，将所有小于它的值从队列中弹出，然后将它插入到队尾，此时队首存放的就是窗口中最大的值。这是因为后面进入窗口的值也会在后面移出窗口，那么如果后面的值比前面的值大，那么前面的值就永远不会成为窗口中的最大值，所以可以把他们从队列中弹出。

class Solution {
public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        deque<int> q;
        vector<int> ans;
        for (int i = 0; i < k; i++) {
            if (q.empty())
                q.push_back(nums[i]);
            else {
                while (!q.empty() && q.back() < nums[i])
                    q.pop_back();
                q.push_back(nums[i]);
            }
        }
        ans.push_back(q.front());
        if (nums[0] == q.front()) q.pop_front();
        for (int i = k; i < nums.size(); i++) {
            if (q.empty())
                q.push_back(nums[i]);
            else {
                while (!q.empty() && q.back() < nums[i])
                    q.pop_back();
                q.push_back(nums[i]);
            }
            ans.push_back(q.front());
            if (nums[i-k+1] == q.front()) q.pop_front();
        }
        return ans;
    }
};

76. 最小覆盖子串

困难

给你一个字符串 s 、一个字符串 t 。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 "" 。

注意：

  • 对于 t 中重复字符，我们寻找的子字符串中该字符数量必须不少于 t 中该字符数量。
  • 如果 s 中存在这样的子串，我们保证它是唯一的答案。

示例 1：

输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"
解释：最小覆盖子串 "BANC" 包含来自字符串 t 的 'A'、'B' 和 'C'。

示例 2：

输入：s = "a", t = "a"
输出："a"
解释：整个字符串 s 是最小覆盖子串。

示例 3:

输入: s = "a", t = "aa"
输出: ""
解释: t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。

提示：

  • m == s.length
  • n == t.length
  • 1 <= m, n <= 105
  • s 和 t 由英文字母组成
滑动窗口

使用两个指针形成一个滑动窗口，在窗口内的字符串如果包含了t中所有字符且出现的次数也满足条件（使用哈希表保存每个字符出现的次数），记录长度、起始和结束的下标，让窗口左边界右移（为了找出更小的满足条件的子串），否则让右边界右移（为了找到一个满足条件的子串）。

class Solution {
public:
    unordered_map<char, int> tmap, smap;
    bool check() {
        for (const auto &p: tmap) {
            if (smap[p.first] < p.second) {
                return false;
            }
        }
        return true;
    }
    string minWindow(string s, string t) {
        int m = s.size(), n = t.size();
        if (m < n) return "";
        for (int i = 0; i < n; i++) tmap[t[i]]++;
        int minLen = 1e5 + 10;
        int st = -1, ed = -1;
        int i = 0, j = 0;
        while (i < m - n + 1 && j < m) {
            smap[s[j]]++;
            bool flag = check();
            if (flag) {
                if (j - i + 1 < minLen) {
                    st = i, ed = j;
                    minLen = j - i + 1;
                }
                smap[s[i]]--;
                smap[s[j]]--;
                i++;
            } else {
                j++;
            }
        }
        if (st == -1 && ed == -1) return "";
        return s.substr(st, ed - st + 1);
    }
};

53. 最大子数组和

中等

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组是数组中的一个连续部分。

示例 1：

输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
输出：6
解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。

示例 2：

输入：nums = [1]
输出：1

示例 3：

输入：nums = [5,4,-1,7,8]
输出：23

提示：

  • 1 <= nums.length <= 105
  • -104 <= nums[i] <= 104
前缀和

因为要求的结果是数组中某一段的和中的一个最大值，所以可以求出前缀和数组，再遍历前缀和数组，初始化最大值ans以及当前最小的数minN为第一个元素，对于每个遍历的数s[i]，求max(ans, max(s[i] - minN, s[i]))，然后更新minN，最后ans就是答案。

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        const int N = 1e5 + 10;
        int ans = 0;
        int n = nums.size();
        int s[N];
        memset(s, 0, N);
        for (int i = 1; i <= n; i++) {
            s[i] = s[i-1] + nums[i-1];
        }
        int minnum = s[1];
        ans = s[1];
        for (int i = 2; i <= n; i++) {
            ans = max(ans, max(s[i] - minnum, s[i]));
            minnum = min(minnum, s[i]);
        }
        return ans;
    }
};
动态规划

计算出以第1个一直到最后一个元素为结尾的最大连续子数组的和，当遍历结束后，得到的就是整个数组中最大连续子数组的和。

怎么计算以某个元素为结尾的最大连续子数组的和呢？如果以前一个数为结尾的最大连续子数组的和大于0，就将其加上当前元素作为最大连续子数组的和，否则将本元素值作为最大连续子数组的和。

class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int ans = nums[0], pre = nums[0];
        for (int i = 1; i < nums.size(); i++) {
            pre = max(nums[i], pre + nums[i]);
            ans = max(ans, pre);
        }
        return ans;
    }
};

56. 合并区间

中等

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回 一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间 。

示例 1：

输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
输出：[[1,6],[8,10],[15,18]]
解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6].

示例 2：

输入：intervals = [[1,4],[4,5]]
输出：[[1,5]]
解释：区间 [1,4] 和 [4,5] 可被视为重叠区间。

提示：

  • 1 <= intervals.length <= 104
  • intervals[i].length == 2
  • 0 <= starti <= endi <= 104
排序

对每个区间按照起点排序，从前往后遍历每个区间，可以分为以下几种情况：

  1. 后一个区间是当前区间的子区间，继续比较下一个区间；
  2. 后一个区间部分与当前区间重叠，改变当前区间的终点为后一个区间的终点，继续比较下一个区间；
  3. 后一个区间与当前区间没有重叠，将当前区间加入到结果中，然后将下一个区间作为当前区间。

当比较结束时，假设当前区间是i，比较区间是j，则可能处于以下两种情况：

  1. j与i有重叠，且j是最后一个区间
  2. j与i没有重叠，且j是最后一个区间

以上两种情况的处理方式是相同的，都只需要把i区间添加到结果中即可。

class Solution {
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        if (intervals.size() == 1) return intervals;
        vector<vector<int>> ans;
        sort(intervals.begin(), intervals.end());
        int i = 0, j = i + 1;
        while (i < intervals.size() && j < intervals.size()) {
            if (intervals[j][0] >= intervals[i][0] && intervals[j][0] <= intervals[i][1]) {
                // 头在前一个区间内
                if (intervals[j][1] >= intervals[i][1]) {
                    intervals[i][1] = intervals[j][1];
                }
                j++;
            } else {
                    vector<int> tmp;
                    tmp.push_back(intervals[i][0]);
                    tmp.push_back(intervals[i][1]);
                    ans.push_back(tmp);
                    i = j;
                    j++;
            }
        }
        if (i < intervals.size()) {
            vector<int> tmp;
            tmp.push_back(intervals[i][0]);
            tmp.push_back(intervals[i][1]);
            ans.push_back(tmp);
        }
        return ans;
    }
};

189. 轮转数组

中等

给定一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。

示例 1:

输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]

示例 2:

输入：nums = [-1,-100,3,99], k = 2
输出：[3,99,-1,-100]
解释: 
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]

提示：

  • 1 <= nums.length <= 105
  • -231 <= nums[i] <= 231 - 1
  • 0 <= k <= 105
辅助数组

由题意可以知道，下标为i的元素轮转后的位置为(i+k)%nums.length，那么可以使用一个数组来保存每个轮转后的元素。

class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        k %= nums.size();
        int n = nums.size();
        vector<int> tmp(n);
        for (int i = 0; i < n; i++) {
            tmp[(i+k)%n] = nums[i];
        }
        nums.assign(tmp.begin(), tmp.end());
    }
};
数组翻转

先对数组进行翻转，原来排在后面的元素翻转后排在前面，轮转k次后，后k个元素在前面，前n-k个元素在后面，先对前k个元素翻转，再对后n-k个元素再翻转即可得到答案。

class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        k %= nums.size();
        reverse(nums.begin(), nums.end());
        reverse(nums.begin(), nums.begin() + k);
        reverse(nums.begin() + k, nums.end());
    }
};

238. 除自身以外数组的乘积

中等

给你一个整数数组 nums，返回 数组 answer ，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积 。

题目数据 保证 数组 nums之中任意元素的全部前缀元素和后缀的乘积都在 32 位 整数范围内。

请 **不要使用除法，**且在 O(*n*) 时间复杂度内完成此题。

示例 1:

输入: nums = [1,2,3,4]
输出: [24,12,8,6]

示例 2:

输入: nums = [-1,1,0,-3,3]
输出: [0,0,9,0,0]

提示：

  • 2 <= nums.length <= 105
  • -30 <= nums[i] <= 30
  • 保证 数组 nums之中任意元素的全部前缀元素和后缀的乘积都在 32 位 整数范围内

**进阶：**你可以在 O(1) 的额外空间复杂度内完成这个题目吗？（ 出于对空间复杂度分析的目的，输出数组 不被视为 额外空间。）

前缀乘积数组和后缀乘积数组

从前往后遍历数组，计算当前元素之前所有元素的乘积，保存到前缀乘积数组中；同样，从后往前遍历，将当前元素后面的所有元素的乘积保存到后缀乘积数组中。前缀乘积数组和后缀乘积数组对应下标的值相乘就是除了该元素所有元素的乘积。

class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        vector<int> ans;
        const int N = 1e5 + 10;
        int prefix[N], suffix[N];
        memset(prefix, 0 ,N);
        memset(suffix, 0 ,N);
        prefix[0] = 1;
        suffix[nums.size()-1] = 1;
        for (int i = 1; i < nums.size(); i++) {
            prefix[i] = prefix[i-1] * nums[i-1];            
        }
        for (int i = nums.size() - 2; i >= 0; i--) {
            suffix[i] = suffix[i+1] * nums[i+1];            
        }
        for (int i = 0; i < nums.size(); i++) {
            ans.push_back(prefix[i] * suffix[i]);         
        }
        return ans;
    }
};
优化空间为O(1)

计算完前缀乘积数组后，从后往前遍历时，直接将前缀乘积数组中对应下标的元素再乘以后面所有元素的乘积，最后前缀乘积数组就是结果。

class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> prefix(n, 1);
        for (int i = 1; i < n; i++) {
            prefix[i] = prefix[i-1] * nums[i-1];            
        }
        for (int i = n - 2, s = nums[n-1]; i >= 0; i--) {
            prefix[i] *= s;
            s *= nums[i];   
        }
        return prefix;
    }
};
