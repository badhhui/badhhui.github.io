Python常见格式化输出方法

Python格式化输出

如果Python程序中想要让输出格式化，比如在一个字符串的任意位置插入一个变量或者让某个变量输出的时候保留n位小数，可以有以下几种方法：

在字符串内插入变量

  1. 在字符串开头的引号前添加 f 或 F 。在这种字符串中，可以在 { 和 } 字符之间输入引用的变量，或字面值的 Python 表达式。

    name = "Bob"
age = 25
print(f'My name is {name} and I am {age} years old.')
# 以上代码的运行结果如下：
>> My name is Bob and I am 25 years old.

  2. 使用str.format()方法，这是一种较新的方法，提供了更多的灵活性。

    name = "Bob"
age = 25
print('My name is {} and I am {} years old.'.format(name, age))
print('My name is {0} and I am {1} years old. Hello, {0}!'.format(name, age))
print('My name is {a} and I am {b} years old.'.format(a=name, b=age))
# 以上代码的运行结果如下：
>> My name is Bob and I am 25 years old.
>> My name is Bob and I am 25 years old. Hello, Bob!
>> My name is Bob and I am 25 years old.

  3. 使用百分号（%）格式化，这是一种较旧的方法，受到C语言的printf函数的影响。

    name = "Bob"
age = 25
print('My name is %s and I am %d years old.' % (name, age))
# 以上代码的运行结果如下：
>> My name is Bob and I am 25 years old.

  4. 在print内使用多个参数

    name = "Bob"
age = 25
print('My name is', name, 'and I am', age, 'years old.')
print('My name is', name, 'and I am', age, 'years old.', sep='')
print('My name is', name, 'and I am', age, 'years old.', sep='#')
# 以上代码的运行结果如下：
>> My name is Bob and I am 25 years old.
>> My name isBoband I am25years old.
>> My name is#Bob#and I am#25#years old.
控制输出精度

  1. 使用百分号（%）格式化

    在%操作符后面使用.数字f的形式来控制浮点数的小数位数。

    pi = 3.1415926
print('%.2f' % pi)  				# 输出：3.14

  2. 使用str.format()方法

    在花括号{}内部，使用冒号:后跟.数字f来指定精度。

    pi = 3.1415926
print('{:.3f}'.format(pi))  		# 输出：3.142

  3. 使用f-strings（格式化字符串字面量）

    在表达式后面使用冒号:，然后是.数字f来控制精度。

    pi = 3.1415926
print(f'{pi:.4f}')  				# 输出：3.1416

  4. 控制整数位数或总位数

    不仅可以控制小数位数，还可以控制整数位数或字符串的总长度，以及它们的对齐方式。

    • %格式化中，可以通过%数字d来指定整数的最小位数，数字前可以添加一个0来表示用零填充。

      number = 7
print("%5d" % number)   		# 输出：    7
print("%05d" % number)  		# 输出：00007

    • 在str.format()和f-strings中，可以使用:数字来控制总长度，并通过在数字前添加一个填充字符和对齐标识（<，>，^）来控制对齐和填充。

      number = 7
print("{:04}".format(number))  	# 输出：0007
print(f"{number:06}")  			# 输出：000007
print(f"{number:<07}")			# 输出：7000000
print(f"{number:>07}")			# 输出：0000007
print(f"{number:^07}")			# 输出：0007000

  5. 格式化百分比

    对于百分比的格式化，可以使用%或f-strings将数值转换为百分比形式，并控制小数位数。

    percent = 0.123456
print("%.2f%%" % (percent * 100))  	# 输出：12.35%
print(f"{percent:.2%}")  			# 输出：12.35%
